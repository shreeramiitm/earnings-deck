Simple Monte Carlo for VaR (Python)

import numpy as np

np.random.seed(42)
mu, sigma = 0.0006, 0.012  # daily drift and volatility
paths = 100_000
horizon = 10  # days

r = np.random.normal(mu, sigma, size=(paths, horizon))
portfolio_pnl = 1_000_000 * (np.exp(r.sum(axis=1)) - 1)

var_95 = np.percentile(portfolio_pnl, 5)
print(f"10-day 95% VaR: ${-var_95:,.0f}")

Financial Math

Discounted Cash Flow (DCF) present value:

\[ PV = \sum_{t=1}^{T} \frac{FCF_t}{(1 + r)^t} \]

CAPM cost of equity:

\[ r_e = r_f + \beta (r_m - r_f) \]
